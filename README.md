# Advanced Wealth Tracker

დავალების პირობის თანახმად, კოეფიციენტი K გამოითვლება შემდეგი ფორმულით:
**K = (სახელის ასოების რაოდენობა + გვარის ასოების რაოდენობა) / დაბადების რიცხვი**

ჩემს შემთხვევაში:
* სახელი: **Elisabed** (8 ასო)
* გვარი: **დვალიშვილი** (10 ქართული ასო)
* დაბადების რიცხვი: **11**

**გამოთვლა:**
K = (8 + 10) / 11 = 18 / 11 ≈ **1.64**

## ბიზნეს ლოგიკა
აპლიკაცია იყენებს `WealthManager.kt` კლასს გამოთვლებისთვის:
`Final Savings = (Income - Expenses) * 1.64`
